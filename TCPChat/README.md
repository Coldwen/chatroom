# TCPChat

## **系统功能分析**

### **主要功能：**

### **TCP_Server：**

1. 创建通信的Socket，监听客户端连接；
2. 给客户端发送、接收消息（文字、文件、窗口震动）。

### **TCP_Client：**

1. 创建通信的Socket，连接服务器；
2. 接收服务器发来的消息（文字、文件、窗口震动）。
