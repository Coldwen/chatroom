# UDPChat
基于C# .NET的WinForm三层架构的UDP通信多功能聊天室。

## 系统功能分析

### 主要功能：

​	1、登录注册；
​	2、查看好友离线在线情况；
​	3、对指定好友发送消息（文字、文件、窗口震动）；
​	4、修改资料。
